## Getting Started

First, install the dependencies:

```bash
npm installl
# or
yarn install

```
Then, run the development server:
```bash
npm run dev
# or
yarn dev
```

To execute the build use the commands:

```bash
npm run build
npm run start
# or
yarn build
yarn start
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
